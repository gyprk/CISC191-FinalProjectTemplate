CISC 191 Final Project

The project acts as a library management system --

View the Digital Library
Add, remove, and rename shelves
Add and remove books from corresponding shelves
Search for books within the library
Save library state, and able to pull the exact state on relaunch


Utilizes the following modules --

Module 1: Interactive menu for actions on arrays.
Module 2: Actions on 2D arrays (shelves and books).
Module 3: Interactive terminal menu for user interaction.
Module 4: Encapsulation to restrict direct data access.
Module 5: Save and load library data with I/O streams.
Module 6: Communication between ClientApp and ServerApp.
Module 7: ArrayList and LinkedList for managing shelves and books.
Module 8: LinkedLists and recursion for searching through library data.
Module 9: Searching functionality for books and shelves.
Module 11: Server to handle multiple clientsApps.
Module 12: Data filtering.


How to use --

First -- Run ServerApp to start the local server.
Then -- Run ClientApp to interact with the library.
