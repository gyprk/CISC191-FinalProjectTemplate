CISC 191 Final Project

The project acts as a library managment system --
-View the Digital Library
-Add, remove, and rename shelves
-Add and remove books from corresponding shelves
-Save library, and able to pull exact state

Utilizes the following modules --
-Module 3: By using a interactive terminal menu
-Module 4: By wrapping text to only access from within class
-Module 5: Allows for saving and loading of librarydata from GwanLibrary.txt
-Module 6: Allows for information to pass from ClientApp and ServerApp

How to use --
-Firstly run ServerApp to run the local server
-Then run ClientApp to interact with the library
